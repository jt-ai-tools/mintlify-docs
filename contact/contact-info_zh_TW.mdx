## 我想回報錯誤 (Bug) 或提交功能請求 (Feature Request)

哇，感謝您的幫忙！您可以 [在 GitHub 中建立一個新議題 (Issue)](https://github.com/lightdash/lightdash/issues)，團隊會儘快回覆您 <Icon icon="face-smile" iconType="solid" />

## 我想對產品提供一些建議回饋

有兩種方式可以提供回饋：

1. 在 GitHub 上建立 [新議題 (New Issue)](https://github.com/lightdash/lightdash/issues)
2. 發送電子郵件至 [support@lightdash.com](mailto:support@lightdash.com)

## 我只想關注你們的最新動態

這真是太棒了 <Icon icon="face-smile-beam" iconType="solid" />

有四種方式可以掌握 Lightdash 的一切：

1. 加入 [Lightdash 社群 Slack](https://join.slack.com/t/lightdash-community/shared_invite/zt-3pbwqmq5e-vhTc7HHcS787w618ngoPHA)。
2. 在我們的 [GitHub 儲存庫](https://github.com/lightdash/lightdash) 上點擊 Star。

<Frame>
  ![](/images/contact/star_github_action-065ae77ee1f786df7b070227514b6cde.png)
</Frame>

3. 在官網首頁底部 [訂閱我們的電子報](https://www.lightdash.com/#wf-form-email-form)（我們會定期發送所有有趣的新動態！）
4. 在 dbt 的 Slack 空間加入 [#tools-lightdash](https://getdbt.slack.com/archives/C026WJE4A69) 頻道。

如果您還不是 dbt Slack 的成員，可以 [在此加入社群](https://www.getdbt.com/community/join-the-community)。加入後，請在 [#tools-lightdash](https://getdbt.slack.com/archives/C026WJE4A69) 頻道與我們交流。

## 我需要存取我的發票 (Invoices)

組織管理員可以請求連結以存取其帳單發票。要獲取發票連結，請透過 [support@lightdash.com](mailto:support@lightdash.com) 與我們聯繫，我們會將連結發送給您。

## 我在設置或使用 Lightdash 時遇到問題

我們明白，重複的 `error` 訊息可能會導致 [憤怒點擊 (Rage Clicking)](https://www.fullstory.com/blog/rage-clicks-turn-analytics-into-actionable-insights)，而憤怒點擊絕非令人愉快的事情。

因此，我們隨時準備提供協助。

首先，[查看我們的常見問題 (FAQs)](https://github.com/lightdash/lightdash/discussions/categories/q-a)，看看是否已經有人找到了您問題的解決方案。

如果您在那裡找不到答案，聯絡我們的最佳方式有：

1. [在我們的 Slack 社群中](https://join.slack.com/t/lightdash-community/shared_invite/zt-3pbwqmq5e-vhTc7HHcS787w618ngoPHA) 發佈您的問題
2. 在 Lightdash 應用程式中向我們發送訊息（點擊導覽列中的問號圖示）
3. 在我們的 [問答區 (Q&A)](https://github.com/lightdash/lightdash/discussions/categories/q-a) 發佈問題
4. [建立新的 GitHub 議題 (GitHub issue)](https://github.com/lightdash/lightdash/issues)（如果是錯誤）
5. 發送電子郵件至 [support@lightdash.com](mailto:support@lightdash.com)
