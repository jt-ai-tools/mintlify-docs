## 瀏覽版本歷史

要造訪版本歷史，請在檢視或編輯圖表時，點擊三點選單下的 `Version History` 選項。

<Frame>
  <img src="/images/guides/version-history/version-history.png"/>
</Frame>

這會帶您進入該圖表的版本歷史頁面，您可以在此看到圖表的先前版本，包括是誰在何時更新了這些版本。您可以點擊任何一個先前版本來進行預覽。

<Frame>
  <img src="/images/guides/version-history/version-history-explorer-ba442ad1aba9dbbef26fd2b7eff21eef.png"/>
</Frame>

找到滿意的版本後，您可以點擊該版本旁邊的三點選單，選擇將圖表恢復到該版本。

<Note>
Lightdash 會儲存圖表的所有疊代版本，因此恢復到先前版本不會刪除任何現有的歷史記錄。
</Note>
<Frame>
  <img src="/images/guides/version-history/version-history-selection-9a266ef03efb84ab3567e1ac80eeedab.png"/>
</Frame>


### 找不到某些版本？

版本歷史的保留期限取決於您使用的 Lightdash Cloud 方案層級：

* Cloud Pro：30 天
* Cloud Enterprise：60 天

如果您有興趣升級方案以獲得更長的歷史記錄，請與團隊聯繫。
