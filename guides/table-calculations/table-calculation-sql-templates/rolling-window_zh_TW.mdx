title: "滾動視窗 (Rolling window)"
description: 滾動視窗（或稱回溯視窗）使用當前行結合前 N 行來計算滾動平均值等指標。這對於平滑波動較大的時間序列數據，以便更好地理解長期趨勢非常有用。

以及用於該表格計算的 SQL：

```sql
SUM(${dbt_orders.count_distinct_order_id}) OVER (
  ORDER BY
    ${dbt_orders.order_date_week}
  ROWS BETWEEN 3 PRECEDING AND CURRENT ROW
) /
  SUM(${dbt_orders.count_distinct_user_id}) OVER (
    ORDER BY
      ${dbt_orders.order_date_week}
    ROWS BETWEEN 3 PRECEDING AND CURRENT ROW
  )
```

計算滾動視窗的 SQL 包含至少兩個部分（以及一個可選的第三部分）：

* `first_column_I_want_to_sum` (第一個要加總的欄位) 是包含您想累加數值的欄位（有時這是一個平均值）。
* `column_I_want_to_order_by` (我想據以排序的欄位) 是您在定義回溯視窗時想要排序的欄位。
* `N` 是您想要包含在計算中的前幾行數量（別忘了當前行也會包含在內，因此對於四週的回溯視窗，我們使用 N=3）。
* `second_column_I_want_to_sum` (第二個要加總的欄位) 是選填的，即另一個您想累加以與第一個欄位進行比較的欄位，例如計算平均值時（就像我們在本例中所做的）。

### 這裡是可以複製貼上用來計算滾動視窗的 SQL

請確保替換欄位並為 `N` 選擇一個數值。

```sql
SUM(${table.column_I_want_to_sum}) OVER (
  ORDER BY
    ${table.column_I_want_to_order_by}
  ROWS BETWEEN N PRECEDING AND CURRENT ROW
) /
  SUM(${table.second_column_I_want_to_sum}) OVER (
    ORDER BY
      ${table.column_I_want_to_order_by}
    ROWS BETWEEN N PRECEDING AND CURRENT ROW
  )
```
