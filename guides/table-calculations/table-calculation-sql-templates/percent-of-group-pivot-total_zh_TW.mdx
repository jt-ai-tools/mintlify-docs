以及用於該表格計算的 SQL：

```sql
${orders.total_order_amount} /
  SUM(${orders.total_order_amount}) OVER (
    PARTITION BY
      ${orders.order_date_month}
  )
```

一般來說，計算佔總額百分比的 SQL 包含兩個關鍵欄位：

* `column_i_want_to_see_the_percent_of_total` (我想查看百分比的欄位) \- 這是您想查看其佔總額比例的欄位。在上述範例中，它是 `sum of profit` (利潤總額)。
* `column_i_want_to_group_by` (我想據以分組的欄位) \- 這是您希望總額按其分組的欄位。在上述範例中，它是 `order date - month` (訂單月份)。

### 這裡是可以複製貼上用來計算佔群組/樞紐總額百分比的 SQL

```sql
${table.column_i_want_to_see_the_percent_of_total} /
  SUM(${table.column_i_want_to_see_the_percent_of_total}) OVER (
    PARTITION BY
      ${table.column_i_want_to_group_by}
  )
```

### 請確保為您的計算添加「百分比」格式

在 `Format` (格式) 標籤中，請務必將格式更新為 `Percent` (百分比)，以便您的表格計算以百分比值（而非數字）顯示。
