使用鍵盤快速鍵在 Lightdash 中更高效地導覽與工作。

<Tip>
  **如何使用快速鍵**

  - 在 Mac 上，請使用 `Cmd` 組合鍵。
  - 在 Windows 和 Linux 上，請使用 `Ctrl` 組合鍵。
</Tip>

## 全域快速鍵

此快速鍵適用於 Lightdash 應用程式的所有位置。

| 快速鍵 | 動作 | 描述 |
| ------------ | ------------- | ----------------------------------------------------------------------- |
| Cmd/Ctrl + K | 全域搜尋 | 開啟全域搜尋，快速尋找圖表、儀表板和探索 (Explores)。 |

## 從表格查詢 (Query from tables) 快速鍵

這些快速鍵在建立新的表格查詢或編輯儲存圖表時可用。

| 快速鍵 | 動作 | 描述 |
| ------------------------------------ | --------------- | ------------------------------------------------------------------------ |
| Cmd + Return<br />Ctrl + Enter | 執行查詢 | 執行當前查詢以查看結果。 |
| Cmd + Option + k<br />Ctrl + Alt + k | 清除探索 | 重置當前探索中的所有選取項與篩選器。 |
| Cmd/Ctrl + c | 複製表格儲存格 | 選取表格儲存格時，將內容複製到剪貼簿。 |
