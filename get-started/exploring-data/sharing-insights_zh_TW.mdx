在 Lightdash 中，主要有三種方式可以與團隊分享見解：

* 分享「進行中」數據探索的 URL
* 分享已儲存圖表的連結
* 下載結果為 .csv 或下載圖表圖片為 .png


### 分享草稿 URL

當您在 Lightdash 中探索數據時，URL 包含了一個指向您目前進度的唯一連結。

在探索數據時複製並貼上該 URL，任何擁有 Lightdash 專案存取權限的人都可以進入探索視圖，並看到與您完全相同的進度。有時，如果您的圖表中包含大量資訊，連結會變得非常長，因此您可以使用內建的連結縮短工具來取得短連結：

### 分享已儲存的圖表

##### 儲存您的圖表

您可以透過點擊圖表面板上的「Save chart (儲存圖表)」按鈕來建立新的儲存圖表或更新現有的圖表。您可以選擇[儲存到空間 (Space) 或儀表板 (Dashboard)](/get-started/exploring-data/using-explores_zh_TW#儲存您的圖表)。

<Frame>
  <img src="/images/get-started/exploring-data/sharing-insights/save-chart.png" alt=""/>
</Frame>


##### 分享您的圖表

專案中的每個人都可以存取已儲存的圖表（只要他們擁有您存放該圖表的空間的存取權限）。他們只需點擊 **Browse (瀏覽) -> All saved charts (所有儲存圖表)** 即可。您也可以直接將 URL 發送給他們。

<Frame>
  <img src="/images/get-started/exploring-data/sharing-insights/saved-charts-browse.png" alt=""/>
</Frame>


### 下載結果或圖表圖片

您可以將圖表下載為 JPEG、PNG、SVG、PDF 或 JSON 格式，並可以選擇不透明或透明背景。您也可以透過點擊圖表上的導出圖示來選擇下載檔案或複製到剪貼簿。

<Frame>
  <img src="/images/get-started/exploring-data/sharing-insights/export-chart.png" alt=""/>
</Frame>

您還可以將結果表格下載為 CSV、Excel 試算表，或將其導出到 Google 試算表（如果您的組織已進行相關設置）。
